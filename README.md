steps to create backend using nodeJS

1. generate package.json
        npm init -y

2. install and import Express module

3. create HTTP server and assign port number

4. create API(application program interface)
    API contains Routes
    syntax:
        app.get,app.put,app.delete,app.post

    make HTTP get req--------------> app.get('path',req_handler)

    make HTTP post req------------->app.post('path',req_handler)

    make HTTP put req------------->app.put('path',req_handler)

    same for delete

5. prepare client req using POSTMAN and REST CLIENT
